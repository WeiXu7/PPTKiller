# PPTKiller ToDo

当前 MVP 已完成：用户登录/注册、项目与历史会话、Harness Agent 流程、资料解析、联网资料/文献/图片检索、DeepSeek 大纲生成、演讲稿、专业 PPTX 导出、真实导出缩略图预览。

## P0：把 MVP 做到可真实试用

- [ ] Web 端逐页编辑器
  - 编辑标题、要点、讲稿备注、版式类型。
  - 支持逐页保存到 session artifact。
  - 编辑后可重新导出 PPTX。

- [ ] 图片替换与分配
  - 每页可选择检索图片、上传图片或不使用图片。
  - 保留图片作者、来源和授权信息。
  - 导出时按页应用用户选择，而不是仅自动轮换。

- [ ] 图表/数据页
  - 从 Excel/CSV 抽取表格字段。
  - 支持柱状图、折线图、表格页、关键指标页。
  - 图表页保留数据来源引用。

- [ ] 导出 QA 面板
  - 在前端显示每页真实渲染缩略图、导出时间、页数、文件状态。
  - 展示导出警告，例如缺引用、缺图片、图片下载失败、OCR 未执行。
  - 支持下载 PPTX 和查看总览图 montage。

## P1：增强 Agent 与资料理解

- [ ] OCR / 视觉理解
  - 对扫描 PDF、图片型 PPT、无描述图片按需解析。
  - 避免对有文本层文档重复 OCR。
  - 需要选型：本地 OCR、云 OCR，或多模态模型。

- [ ] 文献质量分层
  - 区分学术论文、行业报告、网页资料、新闻。
  - 标记 DOI、年份、作者、期刊/会议、访问链接。
  - 对不可靠来源给出降权或替代建议。

- [ ] Agent 事件流
  - 后端将长任务拆成异步 job。
  - 前端实时展示步骤进度，而不是等待接口返回。
  - 可选 Redis/Celery/RQ 或 FastAPI background task。

- [ ] 会话恢复与版本管理
  - 每次修改保留版本。
  - 支持回滚到某次大纲、某次编辑或某次导出。

## P2：协作与产品化

- [ ] 团队分享
  - 分享只读预览链接。
  - 权限控制：私有、链接可见、团队可见。
  - 后续可接入评论/批注。

- [ ] 模板中心
  - 内置商务汇报、研究报告、融资路演、课程培训等模板。
  - 模板影响叙事结构、版式比例、视觉风格。

- [ ] 用户资产库
  - 保存常用品牌色、Logo、字体偏好。
  - 保存历史上传图片与常用引用资料。

- [ ] 生产部署配置
  - PostgreSQL 替代 SQLite。
  - 对象存储保存上传文件和导出产物。
  - 后端日志、错误监控、任务重试、限流。
